### 安装
```
git clone https://github.com/Alex-fun/framework7-webpack-demo.git

cd framework7-webpack-demo

npm install 

```

### 开发

```
gulp build-dev
```

若要代理，请看gulpfile.js中的备注


### 打包

```
gulp build
```
